x86-64 assembly/shellcode cheatsheet

(i) General information and Register
How to test your assembly/shellcode?

i. write code in a *.s file

ii.

nasm *.s
for generating hex code for shellcode, file * generated.

iii.

nasm -f [arch] *.s
for generating file including header info, file *.o generated.

iv. if you want to test your code, linking should be the next step.

ld *.o
type ./a.out to run your code.
